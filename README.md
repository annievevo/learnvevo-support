# LearnVevo – Web Support

Pages statiques pour l'App Store.

## URLs finales (après activation GitHub Pages)
- Support : `https://annievevo.github.io/learnvevo-support/`
- Confidentialité : `https://annievevo.github.io/learnvevo-support/privacy.html`
- CGU : `https://annievevo.github.io/learnvevo-support/legal.html`

## Activer GitHub Pages
1. Créer un repo public `learnvevo-support` sur github.com
2. Pousser ces fichiers
3. Settings → Pages → Source: main branch / root
4. Attendre 2 min → les URLs sont actives
